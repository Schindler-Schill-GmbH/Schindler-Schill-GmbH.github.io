      Properties

      Control Gets the control where the sensor listens to mouse events
      Disposed Gets if the sensor has already been
      IsSupsended Gets if the sensor is currently suspended
      Items Gets the collection of items this sensor affects. Sensing can be limitated by the [!:ItemsLimit] property
      ItemsSource Gets or ests the source of items what limits the sensing. If collection is null, all items on the Items property will be sensed.
      PanelLimit Gets or sets the Panel that will be the limit to be sensed. If set to null, all panels in the Panels property will be sensed.
      Panels Gets the collection of panels this sensor affects. Sensing can be limitated by the PanelLimit property
      Ribbon Gets the ribbon this sensor responds to
      TabLimit Gets or sets the Tab that will be the only to be sensed. If set to null, all tabs in the Tabs property will be sensed.
      Tabs Gets the collection of tabs this sensor affects. Sensing can be limitated by the TabLimit property
