      Properties

      AutoSize Gets or sets a value that indicates whether the ToolTip resizes based on its text. true if the ToolTip automatically resizes based on its text; otherwise, false. The default is true.
      BackColor Gets or sets the background color for the ToolTip.
      BorderColor Gets or sets the border color for the ToolTip.
      ForeColor Gets or sets the foreground color for the ToolTip.
      OwnerDraw Gets or sets a value indicating whether the ToolTip is drawn by the operating system or by code that you provide. If true, the properties 'ToolTipIcon' and 'ToolTipTitle' will set to their default values and the image will display in ToolTip otherwise only text will display.
      Size Gets or sets the size of the ToolTip. Valid only if the Property 'AutoSize' is false.
      ToolTipIcon Gets or sets a value that defines the type of icon to be displayed alongside the ToolTip text. Cannot set if the property 'OwnerDraw' is true.
      ToolTipImage Gets or sets the image for the ToolTip window. Cannot set if the property 'OwnerDraw' is true.
      ToolTipTitle Gets or sets a title for the ToolTip window. Cannot set if the property 'OwnerDraw' is true.
